# Overview

This project analyses financial literacy among young adults in Ireland using data from the OECD/INFE International Survey of Adult Financial Literacy. It combines an interactive Excel dashboard with a companion Python analysis to explore how financial knowledge, behaviour, and attitudes compare across different demographic groups.

The primary deliverable is an Excel dashboard designed to communicate key insights through interactive visualisations. The accompanying Jupyter Notebook recreates and extends the analysis using Python, producing publication-quality charts and validating the calculations used throughout the dashboard.

The project demostrates exploratory data analysis, dashboard design, and data visualisation while highlighting important trends in financial literacy among young Irish adults.

# Project Objectives

This project aims to answer several key questions:

1. How financially literate are young adults (18-29) in Ireland?
2. How do financial knowledge, behaviour, and attitudes compare with OECD averages?
3. What insights can support future financial eduaction initiatives?
